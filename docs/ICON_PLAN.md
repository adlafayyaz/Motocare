# Icon Plan

## Android Icon Source
- Use Android Studio Vector Asset.
- Use Material Icons as vector drawable XML.
- Store in `app/src/main/res/drawable`.
- Use `ImageView` or `MaterialButton` icon.
- Do not use PNG for simple icons.

## Suggested Icons
- Home: `home`
- Motor: `two_wheeler`
- Catat/add: `add`
- Riwayat: `history`
- Profil: `person`
- Servis: `build` or `construction`
- Oli: `opacity` or `water_drop`
- Pajak: `receipt_long` or `article`
- Bensin: `local_gas_station`
- Pengeluaran: `donut_large` or `pie_chart`
- Kilometer: `speed`
- Edit: `edit`
- Hapus: `delete`
- Settings: `settings`
- Backup/export: `file_download` or `ios_share`
- Import: `file_upload`
- About: `info`
- Login Google button: use text plus Google sign-in branding if allowed by Firebase UI.
